### Descrição
Este projeto é uma aplicação Windows Forms em C# que permite uma buscar rapida de palavras em arquivos .txt fazendo indexação de varios arquivos txt em um diretorio.

### Screenshot
![Screen](./Images/tela.png)

### Instalação / Dependências
Para executar este projeto, você precisa ter instalado no seu computador:
- Visual Studio 2022
- Libraria Lucene.net
- Libraria MaterialSkin

### Contribuição
1. Faça um fork do repositório.
2. Crie uma branch com o nome da sua feature (git checkout -b feature/nome).
3. Faça as alterações no código.
4. Faça um commit das suas alterações (git commit -m 'Adiciona a feature nome').
5. Faça um push da sua branch para o seu fork (git push origin feature/nome).
6. Abra um pull request para a branch main deste repositório.
